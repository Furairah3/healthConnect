How AJAX Works in This Demo

This dashboard demonstrates AJAX (Asynchronous JavaScript and XML) which allows web pages to request data from a server without refreshing the page. Here's how it works:

1

User Interaction

You click a button (Load Countries or Load Cities) which triggers a JavaScript function.

2

AJAX Request

JavaScript sends an HTTP request to fetch African data in the background.

3

Server Response

The data (countries or cities) returns without interrupting your experience.

4

Update Content

JavaScript processes the response and updates only the content area dynamically.
